# Move+ Fitness App

Move+ is a gamified fitness tracker built with Flutter.  
It rewards users with Energy Points based on movement, challenges, and engagement.

## Features
- Walk/Run/Cycling tracking
- Movement-based challenges
- Raffle system with Energy entry
- Subscription system with 7-day trial
- Creator & invite rewards
- Supabase backend integration
- Anti-cheat protection

## Tech Stack
- Flutter
- Supabase
- Google Location & Activity Recognition
- Mapbox
- Hive (local storage)
- Dart

## Note
Environment keys have been removed for security reasons.
